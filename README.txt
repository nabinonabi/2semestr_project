# Приложение: Магазин товаров для кемпинга "Camping Shop"

## Описание сути приложения: 
Суть приложения заключается в создании веб-сервиса для магазина товаров для кемпинга. На текущем этапе реализована базовая маршрутизация и вывод текстовой информации.

## Список страниц магазина
В приложении настроены следующие страницы:

1.  **Главная страница** (`/`) — содержит приветствие и навигационные ссылки на остальные разделы приложения.
2.  **Страница об авторе** (`/author/`) — содержит информацию о студенте, выполнившем лабораторную работу 
3.  **Страница о магазине** (`/shop-info/`) — содержит краткое описание тематики магазина
